Template Variables

Add these variables to your template:

Variable Description Example Value
{{to_email}} Recipient email address manager@company.com
{{subject}} Email subject line Weekly Fuel Distribution Report
{{message}} Main email message Your scheduled report is ready
{{report_title}} Report title Fuel Distribution Analysis
{{report_date}} Report generation date 10/6/2025
{{frequency}} Report frequency weekly
